Dz6.GCollector

# Описание

.NET 8 консольное приложение с объектно-ориентированной архитектурой: garbage collector с IDisposable.

# Требования

SDK .NET 8 или новее: https://dotnet.microsoft.com/download
ОС: Windows / Linux / macOS
Желательно: Git 2.4+ для работы с ветками и PR

# Быстрый старт

dotnet build

# Запустить консольное приложение

dotnet run --project ./GCollector
